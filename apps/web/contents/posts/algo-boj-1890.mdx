# 백준 1890번: 점프

N×N 게임판에서 왼쪽 위 칸에서 오른쪽 아래 칸으로 규칙에 맞게 이동할 수 있는 경로의 개수를 구하는 문제입니다.

## 동적 계획법 (DP) 설계

* **상태 정의**: `dp[i][j]` = `(i, j)` 칸에 도달할 수 있는 경로의 개수
* **초기 상태**: `dp[0][0] = 1`
* **점화식**: 현재 칸 `(i, j)`의 값이 `jump`일 때, 오른쪽 `(i, j + jump)`와 아래쪽 `(i + jump, j)`에 `dp[i][j]`의 값을 더해줍니다.

## 전체 코드

경로의 개수가 `2^63 - 1` 이하이므로 자바스크립트에서는 `BigInt`를 사용해야 오차 없이 정답을 출력할 수 있습니다.

```javascript
const fs = require('fs');
const input = fs.readFileSync('/dev/stdin').toString().trim().split('\n');

const N = parseInt(input[0]);
const board = input.slice(1).map(line => line.split(' ').map(Number));

const dp = Array.from({ length: N }, () => Array(N).fill(0n));
dp[0][0] = 1n;

for (let i = 0; i < N; i++) {
  for (let j = 0; j < N; j++) {
    if (dp[i][j] === 0n || (i === N - 1 && j === N - 1)) continue;

    const jump = board[i][j];
    
    // 아래로 점프
    if (i + jump < N) dp[i + jump][j] += dp[i][j];
    // 오른쪽으로 점프
    if (j + jump < N) dp[i][j + jump] += dp[i][j];
  }
}

console.log(dp[N - 1][N - 1].toString());
```

주의점: 자바스크립트의 기본 `Number` 타입은 안전한 정수의 최댓값을 넘어가면 정확도를 잃기 때문에 반드시 `BigInt` 처리가 필요합니다!
