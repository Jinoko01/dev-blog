# Hello World

Welcome to my brand new developer blog built with Next.js 16 (App Router) and Tailwind CSS 4.

## Code Highlighting

One of the essential features is code highlighting. Let me show you an example using `rehype-pretty-code`:

```tsx
import { useState } from 'react';

export default function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <button onClick={() => setCount(count + 1)}>
      Clicked {count} times
    </button>
  );
}
```
